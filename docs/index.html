    dd-color-contrast

    @function dd-color-contrast($_color, $_dark: #000, $_light: #FFF) { ... }

    Description

    Возвращает контрастный цвет по отношению к $_color Выбор цвета осуществляется из аргмунетов $_dark и $_light

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $_color

    цвет, к которому подбирать констрастный, если передана строка - будет взят цвет из палитры

    		Color or String— none
    $_dark

    темный констрастный цвет, если передана строка - будет взят цвет из палитры

    		Color or String#000
    $_light

    светлый констрастный цвет, если передана строка - будет взят цвет из палитры

    		Color or String#FFF

    Returns

    Color or Null —

    контрастный цвет или null в случае ошибки

    Example

    dd-color-contrast(default, #000, #FFF);

    Requires

    • [function] dd-color-palette
    • [function] dd-color-palette
    • [function] dd-color-palette
